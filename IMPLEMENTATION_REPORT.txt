For End Users:
1. Click 🌐 button in navbar (top right)
2. Language switches instantly
3. Your preference is saved
4. Mark jobs complete with "काम पूरा किया" button

For Developers - Add Translation to Component:
1. Import: import { useLanguage } from '../hooks/useLanguage';
2. Use: const { t } = useLanguage();
3. Display: <h1>{t('home.title')}</h1>
4. Done! Works in Hindi or English automatically

For Developers - Add New Translation:
1. Edit: Frontend/src/services/translations.js
2. Add: en: { section: { key: 'English text' } }
3. Add: hi: { section: { key: 'हिंदी टेक्स्ट' } }
4. Use: t('section.key') in component
5. Done! Changes live immediately
